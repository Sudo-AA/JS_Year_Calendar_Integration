# JS_Year_Calendar_Integration

A lightweight enhancement for legacy **JS Year Calendar**.  
Integrates **real-world event data** into your calendar UI, making it easier to display **holidays, company events, sprints, or custom schedules** in a **compact and readable year view**.

---

## Features

- Fully integrates with **[js-year-calendar](https://github.com/year-calendar/js-year-calendar)**  
- Fetches **public holidays dynamically** via free APIs  
- Displays events with **day, month, and year badges**  
- Compact, flat UI with subtle shadows and borders  
- Supports **multi-line text wrap** for long event names  
- Fully **AJAX-ready** for live updates  

---
